## A Simple Caclulator
This projects aims at building a command line calculator
As of now this project supports
addition
subtraction
division
multiplication
modulus
### usage
python calc.py
action:
add
sub
mul
div
mod
number1 is integer
number2 is interget
Examples
add
'''
python calc.py add 6 5
11
'''
sub
'''
python calc.py sub 6 5
1
'''
mul
python calc.py mul 6 5
30
'''
div
python calc.py div 6 5
1.2
'''
mod
python calc.py mod 6 5
1
